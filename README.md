# Project-3: Time-Series and Forecasting-

##COVID‑19 Time Series Analysis (US)  
--------------------------------------------------------------------------------------------------------------------------------------------
A data science project using Johns Hopkins CSSE COVID‑19 data to analyze confirmed cases and deaths across US counties and states. Includes preprocessing, descriptive analysis, ARIMA and LSTM forecasting, stationarity checks, model evaluation, and insight generation with policy implications.

---

 The project explores the progression of COVID‑19 confirmed cases and deaths across US counties and states using the Johns Hopkins CSSE dataset. The goal is to generate insights into pandemic dynamics, identify high‑risk regions, and evaluate forecasting models for policy relevance.
 
---
All coding and visualization were done in **Jupyter Notebook**, making the workflow transparent and reproducible. The analysis includes data preprocessing, descriptive statistics, and visualizations such as line plots, heatmaps, and choropleth maps to highlight national and regional trends.

---

For forecasting, two models were implemented:
- **ARIMA (AutoRegressive Integrated Moving Average)** — interpretable, statistically grounded, and useful for policy communication.  
- **LSTM (Long Short‑Term Memory neural network)** — capable of capturing complex, non‑linear temporal patterns with higher accuracy.  

The project compares these models, evaluates their performance using error metrics, and applies them to forecast future case and death trajectories. Insights are then connected to external factors such as lockdowns, reopenings, and vaccination rollouts.

By following this workflow, the project demonstrates how time series analysis can be applied to real‑world public health data, combining statistical rigor with machine learning to inform evidence‑based decision making.
